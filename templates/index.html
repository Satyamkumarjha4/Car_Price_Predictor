Student Exam Performance Indicator

A machine learning-powered tool designed to predict math scores based on various student attributes. This project leverages advanced data preprocessing, feature engineering, and predictive modeling techniques to deliver accurate results. Simply input details such as parental education, lunch type, test preparation level, and current scores to receive a prediction. The backend is built using Python and Flask, while the frontend is designed for simplicity and ease of use. This project demonstrates the implementation of an end-to-end machine learning pipeline, including data collection, model training, and deployment.
